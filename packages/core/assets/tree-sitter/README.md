# Vendored Tree-sitter Asset

`tree-sitter-scala.wasm` is built by `tree-sitter-scala@0.24.0` and is vendored because
`@vscode/tree-sitter-wasm@0.3.1` does not ship a Scala grammar.

- Upstream: <https://github.com/tree-sitter/tree-sitter-scala>
- License: MIT (`tree-sitter-scala.LICENSE`)
- SHA-256: `b7ec2bb29c19827abcefd18ed5cb5a43596009f96a5d53c5b9d1f9676d7521c3`

The other supported WASM grammars are loaded from the pinned
`@vscode/tree-sitter-wasm@0.3.1` dependency.
